数据库

发表于 2022-10-31 更新于 2022-11-02

简介

MySQL 是一种数据库。数据库定义了存储信息的结构。
在数据库中，存在着一些表。类似 HTML 表格，数据库表含有行、列以及单元。
查询：查询是一种询问或请求。
通过 MySQL，我们可以向数据库查询具体的信息，并得到返回的记录集。例如：
SELECT LastName FROM Persons

Connect

访问并处理数据库中的数据之前，必须创建到达数据库的连接
语法：
mysql_connect(servername,username,password)
关闭连接：mysql_close() 函数

Create

mysql_query() 函数：为了让 PHP 执行语句，必须使用，用于向 MySQL 连接发送查询或命令
CREATE DATABASE 语句：
用于在 MySQL 中创建数据库。语法：
CREATE DATABASE database_name
例如：mysql_query(“CREATE DATABASE my_db”,$con)
mysql_select_db() 函数选取数据库
CREATE TABLE 用于在 MySQL 中创建数据库表。
语法：
CREATE TABLE table_name
(column_name1 data_type,
column_name2 data_type,
column_name3 data_type,
…….
)
每个表都应有一个主键字段。主键用于对表中的行进行唯一标识。每个主键值在表中必须是唯一的。此外，主键字段不能为空，这是由于数据库引擎需要一个值来对记录进行定位
要确保主键字段不为空，我们必须向该字段添加 NOT NULL 设置。例如：
personID int NOT NULL AUTO_INCREMENT

Insert

INSERT INTO 语句：用于向数据库表添加新记录。
语法：
INSERT INTO table_name
VALUES (value1, value2,….)
如果需要列则
INSERT INTO table_name (column1, column2,…)
VALUES (value1, value2,….)
也可利用超全局变量将表单中的数据加入数据库中

Slect

SELECT 语句：用于从数据库中选取数据。、
语法：
SELECT column_name(s) FROM table_name
mysql_fetch_array() 函数：以数组的形式从记录集返回第一行。每个随后对 mysql_fetch_array() 函数的调用都会返回记录集中的下一行（一行一行地收集，一行一行地记录）
如需在HTML 表格中显示结果，用表格格式即可。

Where

WHERE 子句：如需选取匹配指定条件的数据，请向 SELECT 语句添加 WHERE 子句。
例如：
$result = mysql_query(“SELECT * FROM PersonsWHERE FirstName=’Peter’”)

Order

ORDER BY 关键词用于对记录集中的数据进行排序：
result = mysql_query(“SELECT * FROM Persons ORDER BY age”)
DESC 关键词来设定降序排序：
SELECT column_name(s)
FROM table_name
ORDER BY column_name DESC
可以根据多个列进行排序。当按照多个列进行排序时，只有第一列相同时才使用第二列：
SELECT column_name(s)
FROM table_name
ORDER BY column_name1, column_name2

Update

UPDATE 语句：用于在数据库表中修改数据：
UPDATE table_name
SET column_name = new_value
WHERE column_name = some_value
例如：mysql_query(“UPDATE Persons SET Age = ‘36’
WHERE FirstName = ‘Peter’ AND LastName = ‘Griffin’”)

Delete

DELETE FROM 语句：用于从数据库表中删除记录
DELETE FROM table_name
WHERE column_name = some_value
例如：mysql_query(“DELETE FROM Persons WHERE LastName=’Griffin’”)

ODBC

ODBC 是一种应用程序编程接口（Application Programming Interface，API），使我们有能力连接到某个数据源（比如一个 MS Access 数据库）。
创建 ODBC 连接:
通过一个 ODBC 连接，您可以连接到您的网络中的任何计算机上的任何数据库，只要 ODBC 连接是可用的
odbc_connect() 函数：用于连接到 ODBC 数据源
odbc_exec() 函数：用于执行 SQL 语句：
$conn=odbc_connect('northwind','','');
$sql="SELECT * FROM customers";
$rs=odbc_exec($conn,$sql);
odbc_fetch_row() 函数：用于从结果集中返回记录。如果能够返回行，则返回 true，否则返回 false。
该函数有两个参数：ODBC 结果标识符和可选的行号：
odbc_fetch_row($rs)
odbc_result() 函数：用于从记录中读取字段。该函数有两个参数：ODBC 结果标识符和字段编号或名称
例如：$compname=odbc_result($rs,"CompanyName")
odbc_close()函数：用于关闭 ODBC 连接
